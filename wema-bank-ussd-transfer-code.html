    Wema Bank is one of the leading banks in Nigeria but also one of the most reliable for online transactions. Wema bank transfer code makes it easier for their customers to transfer money,load airtime and pay bills from the comfort of your home. Wema bank transfer code works on all type of phone and all networks.

    Wema Bank transfer code can be used to Open an Account, Generate a PIN, Balance Enquiry, Change PIN, Funds Transfer, Airtime & Data Top-up, Cash on the Go (Cardless Withdrawal),DSTV Subscription, GOTV Subscription, Startimes Subscription, Bill Payment (Electricity Bills), Bill Payment (Smile Network Subscription), Bill Payment (Swift Network Subscription), MCash (Merchant Payment), Bill Payment (LCC Toll) and Receive Western Union Payment.

    What is WemaBankUSSD transfer code

    Wema bank transfer code is *945#.

    To open an accountusing Wema USSD code dial *945*1# and follow the command prompt. Your account number will be sent to you immediately

    To link your BVN to your Wema bank accountnumber dial *945*BVN# for instance *945*0000000000#

    To generate pin for new accounts using Wema USSD code *945*100# then follow the commandprompt.

    For old accounts dial *945*50# then follow the commandprompt

    To check your WemaBankaccountbalance dial *945*0# andfollow the command prompt

    To transfermoneyusing Wema bank transfer code Dial *945*BeneficiaryAccountNumber*Amount# for instance dial *945*00000000000*10000# then follow thecommand prompt

    To load airtimeusing Wema Bank USSD code dial *945*amount# for self and *945*number*amount# forthird party. For instance *945*080300000000*1000#

    For Dstv subscription dial *945*16*smartcard number#

    For GoTVsubscription dial *945*17*smartcard number#

    For startimes subscription dial *945*16*smartcard number*amount#

    For Electricity bill dial *945*19*MeterNumber*Amount#

    For SmileNetworksubscriptiondial*945*20*IDNumber#.

    For mCash dial dial *945*22*Merchantcode*Amount# or the universal code *402*Merchantcode*Amount#.
